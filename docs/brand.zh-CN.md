# 品牌系统

Rootloom 结合两个概念：

- **Root**：仓库事实，包括源码、Schema、测试、指导与根因证据。
- **Loom**：把这些事实组织成范围清晰、经过验证的个人工程修改。

紫色编织根系标记是规范符号。它表达一个完整质量闭环，而不是产品等级或强制代理编排。
