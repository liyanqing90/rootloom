# Rootloom 品牌系统

Rootloom 是产品名，不是品类说明。它能解释项目的工作方式，又不会把产品缩减成某一个功能：

- **Root**：仓库事实、项目指导和根因证据。
- **Loom**：把 Skills、Hooks、Rules、Agents 与验证组成可选能力层的织架。

英文主标语是 **“Reliable Codex engineering, woven from the roots up.”** 中文表达为 **“从根部织起可靠的 Codex 工程。”** 需要补充语义时使用 **“从项目事实到验证交付。”**

## 图形构成

品牌符号由一根连续纤维构成：

1. 小型证据网络形成根系；
2. 五个框架代表五个正式能力层；
3. 受控分支代表可选的 Agent 委派；
4. 唯一的琥珀色勾表示可执行验证。

它刻意不使用字面树形、通用盾牌、机器人、大脑或源自 OpenAI 的符号。Rootloom 是独立开源项目，不得被表述为 OpenAI 官方产品。

## 色彩

| Token | 色值 | 用途 |
| --- | --- | --- |
| Ink | `#12101F` | 深色画布与主要对比 |
| Iris | `#8A78FF` | 能力纤维与主品牌色 |
| Mint | `#58D8B1` | 证据根系与健康状态 |
| Amber | `#FFB45D` | 只表示验证结果 |
| Paper | `#FAF9FF` | 浅色表面与深色模式文字 |

Amber 必须保持稀缺，不用于普通装饰，只表示通过门禁或完成结果。
