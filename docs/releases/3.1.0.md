## Notes

This Minor release narrows material quarantine for environment templates and public certificates while adding an exact Intake-only, seal-bound reviewability declaration. Default Intake remains baseline v3; baseline v4 is emitted only for callers using `--reviewable-path`, and v2/v3/v4 remain readable and sealable. Strong secret forms and declared Sensitive roots remain non-overridable, Finalizer cannot mutate reviewability, and every reviewable declaration still raises security risk.
